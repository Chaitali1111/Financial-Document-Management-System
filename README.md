# Financial Document Management System

AI-powered Financial Document Management System built using FastAPI, JWT Authentication, RBAC and RAG-based Semantic Search.

## Features

- User Authentication using JWT
- Role-Based Access Control (RBAC)
- PDF Upload and Processing
- Text Extraction from PDF
- Chunk Creation
- Vector Embedding Storage
- RAG-based Semantic Search
- FastAPI Backend
- Render Deployment

## Tech Stack

- FastAPI
- Python
- SQLite
- ChromaDB
- JWT Authentication
- Render
